Research

Research Interests: Parallel systems and languages, network of workstations, Beowulf clusters, system hardware/software problems, application development using PVM and MPI, analysis and implementation of parallel algorithms, parallel graph algorithms, and parallel data structures.

Director of the Beowulf Cluster Lab (funded by National Science Foundation Award No. 0321233)

I am also the Chief Scientist for Balihoo, Inc, a media planning search engine software company.

Organizer of the First IEEE Idaho High Performance Computing Workshop.
